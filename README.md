<p>A simple yet interactive Tic Tac Toe game built with vanilla JavaScript, focused on clean logic and dynamic UI updates.

This project demonstrates core JavaScript concepts such as:

DOM manipulation
Event handling
Game state management
Array methods like every() for efficient condition checking

Players take turns marking the grid as X and O, while the game automatically detects winning combinations or a draw. The interface updates in real-time, highlights the winning line, and prevents further moves once the game is over. </p>
